# React Learning - Task Manager App

A comprehensive Task Manager application built to teach all essential React concepts through practical, hands-on learning.

## 🎯 What You'll Learn

This project covers ALL the important React concepts:
- ✅ **Components** - Building reusable UI pieces
- ✅ **Props** - Passing data between components
- ✅ **State (useState)** - Managing component data
- ✅ **Effects (useEffect)** - Side effects and lifecycle
- ✅ **Context (useContext)** - Global state management
- ✅ **Refs (useRef)** - DOM access and persistent values
- ✅ **Memoization (useMemo, useCallback)** - Performance optimization
- ✅ **Custom Hooks** - Reusable logic extraction
- ✅ **Lists & Keys** - Rendering arrays efficiently
- ✅ **Conditional Rendering** - Dynamic UI based on state
- ✅ **Forms** - Controlled components
- ✅ **Event Handling** - User interactions

## 🚀 Quick Start

```bash
# Navigate to the project
cd react-learning-app

# Install dependencies (if not already done)
npm install

# Start the development server
npm run dev
```

Open your browser to http://localhost:5173

## 📁 Project Structure

```
src/
├── components/         # React components
│   ├── Header.jsx     # Header with theme toggle

│   ├── TaskItem.jsx   # Individual task
│   └── TaskList.jsx   # List with filters
├── contexts/          # Context providers
│   └── ThemeContext.jsx
├── hooks/             # Custom hooks
│   ├── useLocalStorage.js
│   └── useToggle.js

```
